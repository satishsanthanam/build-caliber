Chapter 6: We Distribute, Yet Things Multiply

1. The Distributive Property

  • Distributivity relates multiplication and addition: a(b + c) = ab + ac.
  • Increments in Products: If numbers a and b are increased by 1, the new product is (a + 1)(b + 1) = ab + a + b + 1.

2. Algebraic Identities

  • Identities are mathematical statements expressing the equality of two algebraic expressions for all integer values.
  • General Expansion: (a + m)(b + n) = ab + mb + an + mn.
  • This applies to negative integers as well: (a + u)(b - v) = ab - av + ub - uv.
  • Like terms (terms with the exact same letter-numbers) can be added/simplified (e.g., ba + ab = 2ab).

3. Fast Multiplications

  • Distributivity enables quick mental math.
  • Example (Multiply by 11): 3874 × 11 = 3874(10 + 1) = 38740 + 3874 = 42614.
  • Example (Multiply by 101): dcba × 101 = dcba(100 + 1) = dcba00 + dcba.

4. Special Cases of the Distributive Property

These core identities can be visualized geometrically using areas of squares and rectangles.

  • Identity 1A (Square of Sum): (a + b)2 = a2 + 2ab + b2
  • Identity 1B (Square of Difference): (a - b)2 = a2 - 2ab + b2
  • Identity 1C (Difference of Squares): (a + b)(a - b) = a2 - b2
Examples & Applications
  • 652 = (60 + 5)2 = 602 + 2(60)(5) + 52 = 3600 + 600 + 25 = 4225
  • 98 × 102 = (100 - 2)(100 + 2) = 1002 - 22 = 10000 - 4 = 9996

5. Investigating Patterns

  • Algebra helps generalize visual and numerical patterns.
  • Example Pattern: 2(22 + 12) = 32 + 12 generalizes to 2(a2 + b2) = (a + b)2 + (a - b)2.
  • Geometric Sequences: Finding the number of items in step k often involves formulating an algebraic expression like k2 + 2k based on visual growth.
