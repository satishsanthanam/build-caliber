Class 8 (Part 1) — Chapter 6: We Distribute, Yet Things Multiply

6.1 Some Properties of Multiplication

6.1.1 Distributive Property & Increments
  • Distributive Property: Relates multiplication and addition: a(b + c) = ab + ac. This can be visualized geometrically as splitting a grid of a rows and (b + c) columns into two smaller grids.
  • Single Increment: Increasing one number in a product by 1 increases the total product by the value of the other number: a(b + 1) = ab + a.
  • Double Increment: Increasing both numbers in a product by 1 increases the product by a + b + 1: (a + 1)(b + 1) = ab + a + b + 1.
  • Mixed Increment: Increasing one number by 1 and decreasing the other by 1: (a + 1)(b - 1) = ab + b - a - 1.

6.1.2 Algebraic Identities
  • Identity: A mathematical statement expressing the equality of two algebraic expressions that holds true for all possible values of the variables.
  • Identity 1: (a + m)(b + n) = ab + mb + an + mn. Geometrically, this represents the total area of a grid divided into four distinct sub-regions.
6.2 Special Cases of the Distributive Property

6.2.1 Square of the Sum & Difference
  • Identity 1A (Square of Sum): (a + b)2 = a2 + 2ab + b2.
  • Identity 1B (Square of Difference): (a - b)2 = a2 - 2ab + b2.

6.2.2 Difference of Squares
  • Identity 1C: (a + b)(a - b) = a2 - b2.
  • Sridharacharya's Method: A modified form of Identity 1C used to quickly compute squares: a2 = (a + b)(a - b) + b2 (e.g., 312 = (32 × 30) + 12 = 961).
6.3 Mind the Mistake, Mend the Mistake
  • Incorrect Distribution: -3p(-5p + 2q) is often mistakenly written as -3p + 5p - 2q. The correct expansion is 15p2 - 6pq.
  • Binomial Squaring Error: (5m + 6n)2 is NOT equal to 25m2 + 36n2. The middle term 2ab must be included: 25m2 + 60mn + 36n2.
  • Like Terms: Only terms with identical variable parts can be combined (e.g., 5w2 + 6w cannot be simplified to 11w2).
6.4 Pattern Analysis & Historical Context
  • Khaṇḍa-guṇanam: Brahmagupta's term for "multiplication by parts" using the distributive property, detailed in the Brahmasphuṭasiddhānta.
  • Ista-gunana: Ancient Indian methods for fast multiplication using the distributive property.
  • Multiple Representations: Visual patterns of circles or tiles can be interpreted in multiple algebraic ways (e.g., (k + 1)2 - 1, k2 + 2k, or k(k + 2)), all simplifying to the same equivalent expression.
