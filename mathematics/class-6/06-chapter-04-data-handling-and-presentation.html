Chapter 4: Data Handling and Presentation

4.1 Collecting and Organising Data

Any collection of facts, numbers, measures, observations or other descriptions of things that convey information about those things is called data.

Tally Marks and Frequency

To organise data systematically, we can use tally marks. A tally mark $|$ represents one occurrence. When the count reaches $5$, a diagonal line is drawn through the previous four marks to represent a group of five: $\cancel{||||}$.

The number of times a particular value or preference occurs is called its frequency.

4.2 Pictographs

A pictograph represents data through pictures or symbols of objects. It helps in understanding and interpreting data at a quick glance.

Scale or Key in Pictographs

A scale or key is essential to define what each symbol represents. For example:

  • $\odot = 1\text{ student}$
  • $\odot = 10\text{ students}$
  • $\triangle = 10\text{ children}$, where a half triangle represents $5$ children.

Preparing a pictograph can be challenging when the data is large or when frequencies are not exact multiples of the scale.

4.3 Bar Graphs

A bar graph (or column graph) is a visual representation of data using bars of uniform width, drawn horizontally or vertically with equal spacing between them.

  • The length or height of each bar represents the frequency of the corresponding category.
  • A suitable scale must be chosen to represent large frequencies.
4.4 Drawing a Bar Graph

To draw a bar graph, follow these steps:

  1. Draw two perpendicular lines: a horizontal axis and a vertical axis.
  2. Along one axis (usually horizontal), mark the categories with equal spacing.
  3. Along the other axis (usually vertical), mark the frequencies using a chosen scale (e.g., $1\text{ unit length} = 10\text{ units}$).
  4. Draw rectangular bars of uniform width whose heights correspond to the respective frequencies.
4.5 Artistic and Aesthetic Considerations

Visual presentations should be appealing, easy to understand, and fit the intended space. Choosing the correct scale and orientation is crucial.

  • Vertical Bars (Column Graphs): Best suited for representing heights or quantities measured upwards from the ground.
  • Horizontal Bars: Best suited for representing lengths parallel to the ground (e.g., river lengths, distances).

Infographics

Infographics (information graphics) are data visualisations beautified with extensive artistic and visual imagery to attract attention and communicate information quickly.

Caution: While infographics are visually appealing, they can sometimes be misleading if the proportions (such as width and height) are not maintained accurately.

Summary
  • Data is a collection of facts, numbers, measures, or observations.
  • Data can be organised in tabular form using tally marks.
  • Frequencies are the counts of occurrences of values.
  • Pictographs represent data using pictures, with a specified scale.
  • Bar graphs use bars of uniform width, where height/length represents frequency.
  • Markings of unit lengths on a bar graph scale must always start from zero.
