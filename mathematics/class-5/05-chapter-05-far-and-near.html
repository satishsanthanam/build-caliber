Class 5 — Chapter 5: Far and Near

5.1 Introduction to Measuring Length

We use different standard units to measure length and distance depending on the size of the object or the distance being measured. In this chapter, we explore various units of measurement and the relationships between them.

Choosing Appropriate Units

Identify whether to use Metre (m) or Centimetre (cm) for measuring the following:

  • Height of India Gate: $42\text{ m}$
  • Length of a handkerchief: $40\text{ cm}$
  • Depth of a well: $50\text{ m}$
  • Length of a mobile phone: $13\text{ cm}$
  • Length of an elephant's trunk: $2\text{ m}$
  • Distance between two buttons on a shirt: $5\text{ cm}$

Different Units but Same Measure

Using a double number line, we can visualize the relationship between metres and centimetres:

  • $1\text{ m} = 100\text{ cm}$
  • $2\text{ m} = 200\text{ cm}$
  • $4\text{ m} = 400\text{ cm}$
  • $5\text{ m} = 500\text{ cm}$
  • $7\text{ m} = 700\text{ cm}$
5.2 Comparing Lengths

Comparing lengths requires converting them to the same unit before applying comparison operators ($<$, $=$, $>$).

Comparison Exercises

  • $456\text{ cm} < 5\text{ m}$ (since $5\text{ m} = 500\text{ cm}$)
  • $55\text{ cm} + 200\text{ cm} > 200\text{ cm} + 54\text{ cm}$
  • $6\text{ m } 5\text{ cm} < 6\text{ m } 50\text{ cm}$
  • $2\text{ m } 150\text{ cm} = 3\text{ m } 50\text{ cm}$ (since $150\text{ cm} = 1\text{ m } 50\text{ cm}$)
  • $238\text{ cm} = 138\text{ cm} + 1\text{ m}$

World's Tallest Statues Comparison

Heights of some of the world's tallest statues:

  • Statue of Unity, India: $182\text{ m}$
  • Spring Temple Buddha, China: $128\text{ m}$
  • Guanyin of Nanshan, China: $108\text{ m}$
  • Statue of Liberty, USA: $93\text{ m}$
  • The Motherland Calls, Russia: $91\text{ m}$
  • Christ the Redeemer, Brazil: $38\text{ m}$
5.3 Measuring Long Distances

For measuring long distances, we use specialized tools and the unit Kilometre (km).

Tools for Measuring Long Distances

  • Long Tape: Used for manual measurements on playgrounds or fields.
  • Rodometer (Measuring Wheel): A wheel that counts rotations to calculate distance walked.
  • Rangefinder: An optical or electronic device to measure distance to an object.
  • Laser Distance Meter: Uses laser beams to measure distances with high precision.

The Kilometre Unit

When we walk $1,000\text{ m}$, we say we have walked $1\text{ km}$.

$$\text{Kilo} = \text{Thousand}$$

$$1,000\text{ m} = 1\text{ km}$$

Rope Lengths to Make 1 km

  • $1,000\text{ m}$ rope: $1$ rope needed
  • $100\text{ m}$ rope: $10$ ropes needed
  • $10\text{ m}$ rope: $100$ ropes needed
  • $200\text{ m}$ rope: $5$ ropes needed
  • $500\text{ m}$ rope: $2$ ropes needed
  • $250\text{ m}$ rope: $4$ ropes needed
5.4 The World of Small Things

To measure very small objects, we divide a centimetre into ten equal parts called Millimetres (mm).

The Millimetre Unit

$$1\text{ cm} = 10\text{ mm}$$

Small marks on a standard ruler represent millimetres, allowing for high precision when measuring small items like sprouts, small screws, or fingernails.

5.5 Relationships and Operations between Units

Understanding conversions is essential for performing arithmetic operations on lengths.

Summary of Unit Relationships

  • $10\text{ mm} = 1\text{ cm}$
  • $100\text{ cm} = 1\text{ m}$
  • $1,000\text{ m} = 1\text{ km}$

Adding and Subtracting Lengths

When adding or subtracting lengths, group similar units together or convert all quantities to the same unit before calculating.

Example (Addition): $3\text{ km } 450\text{ m} + 4\text{ km } 650\text{ m}$

  • Sum of kilometres: $3\text{ km} + 4\text{ km} = 7\text{ km}$
  • Sum of metres: $450\text{ m} + 650\text{ m} = 1,100\text{ m} = 1\text{ km } 100\text{ m}$
  • Total: $7\text{ km} + 1\text{ km } 100\text{ m} = 8\text{ km } 100\text{ m}$

Example (Subtraction): $63\text{ m} - 16\text{ m } 75\text{ cm}$

  • Convert to centimetres: $6,300\text{ cm} - 1,675\text{ cm} = 4,625\text{ cm}$
  • Convert back: $46\text{ m } 25\text{ cm}$

Multiplying and Dividing Lengths

Multiplication Example: If $1\text{ m } 80\text{ cm}$ of cloth is needed for one shirt, how much is needed for $20$ shirts?

$$20 \times 1\text{ m } 80\text{ cm} = (20 \times 1\text{ m}) + (20 \times 80\text{ cm})$$

$$= 20\text{ m} + 1600\text{ cm} = 20\text{ m} + 16\text{ m} = 36\text{ m}$$

Division Example: If a road of $12\text{ km } 600\text{ m}$ is laid equally over $6$ days, how much is laid each day?

$$(12\text{ km } 600\text{ m}) \div 6 = 2\text{ km } 100\text{ m}\text{ per day}$$

5.6 Non-Metric Units: Inches and Feet

In daily life, we also use non-metric units like Inches and Feet to measure height and depth.

Inches and Feet Conversions

  • $1\text{ inch} \approx 2\text{ cm } 5\text{ mm}$
  • $1\text{ foot} = 12\text{ inches} \approx 30\text{ cm}$
