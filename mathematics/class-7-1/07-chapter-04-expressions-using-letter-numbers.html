4.1 The Notion of Letter-Numbers

This section introduces a concise way of expressing mathematical relations and patterns, highlighting how it aids in understanding and explaining these relationships.

Example 1: Shabnam and Aftab's Ages

Shabnam is 3 years older than Aftab. If Aftab's age is 10 years, Shabnam's age is 13 years. If Aftab's age is 18 years, Shabnam's age will be 21 years.

  • To find Shabnam's age given Aftab's age, add 3 to Aftab's age.
  • This can be written as: Shabnam's age = Aftab's age + 3.
  • Mathematical relations are often represented in a shorthand form using symbols (letters or short phrases).
  • Letter-numbers: Letters like $a$ (for Aftab's age) and $s$ (for Shabnam's age) used to represent numbers.
  • Algebraic expressions: Mathematical expressions containing letter-numbers, such as $a + 3$.
  • The expression to find Shabnam's age is $s = a + 3$.
  • If $a = 23$ (Aftab's age), then $s = 23 + 3 = 26$ years.

Finding Aftab's Age from Shabnam's Age

  • Aftab is 3 years younger than Shabnam.
  • Expression: Aftab's age = Shabnam's age $‚Äď 3$.
  • Using letters: $a = s ‚Äď 3$.
  • If Shabnam's age ($s$) is 20, Aftab's age ($a$) is $20 ‚Äď 3 = 17$ years.

Example 2: Parthiv's Matchstick Patterns

Parthiv makes L-shaped patterns with matchsticks. Each L uses 2 matchsticks.

  • Number of matchsticks for 5 Ls: $5 imes 2 = 10$.
  • Number of matchsticks for 7 Ls: $7 imes 2 = 14$.
  • Number of matchsticks for 45 Ls: $45 imes 2 = 90$.
  • The relation: Number of matchsticks = $2 imes$ Number of L's.
  • Using $n$ for the number of L's, the algebraic expression is $2 imes n$ or $2n$.

Example 3: Ketaki's Coconut-Jaggery Laddus

Price of a coconut: ‚āĻ35. Price of 1 kg jaggery: ‚āĻ60.

  • Cost for 10 coconuts and 5 kg jaggery: $10 imes ‚āĻ35 + 5 imes ‚āĻ60 = ‚āĻ350 + ‚āĻ300 = ‚āĻ650$.
  • Cost for 8 coconuts and 9 kg jaggery: $8 imes ‚āĻ35 + 9 imes ‚āĻ60 = ‚āĻ280 + ‚āĻ540 = ‚āĻ820$.
  • To find the total amount for $c$ coconuts and $j$ kg jaggery:
    • Cost of coconuts: $c imes 35$.
    • Cost of jaggery: $j imes 60$.
    • Total amount: $c imes 35 + j imes 60$.
  • If $c = 7$ and $j = 4$, total amount: $7 imes 35 + 4 imes 60 = 245 + 240 = ‚āĻ485$.

Example 4: Perimeters of Simple Shapes

  • The perimeter of a square is 4 times the length of its side.
  • Expression: $4 imes q$, where $q$ is the sidelength.
  • If sidelength $q = 7$ cm, perimeter is $4 imes 7 = 28$ cm.
  • Letter-numbers and algebraic expressions allow expressing general mathematical relations concisely, often called formulas.

4.2 Revisiting Arithmetic Expressions

This section revisits concepts like swapping, grouping, brackets, and the distributive property in arithmetic expressions, which also apply to algebraic expressions.

  • Expressions to evaluate:
    1. $23 - 10 imes 2$
    2. $83 + 28 - 13 + 32$
    3. $34 - 14 + 20$
    4. $42 + 15 - (8 - 7)$
    5. $68 - (18 + 13)$
    6. $7 imes 4 + 9 imes 6$
    7. $20 + 8 imes (16 - 6)$
  • Evaluation of $23 - 10 imes 2$:

    Terms are $23$ and $-10 imes 2$.

    $23 + (-10 imes 2) = 23 + (-20) = 3$.

  • Evaluation of $83 + 28 - 13 + 32$:

    Group terms for easier calculation:

    $(83 + 28 + 32) - 13 = 143 - 13 = 130$.

    Alternatively, $(83 - 13) + (28 + 32) = 70 + 60 = 130$.

  • Evaluation of $68 - (18 + 13)$:

    Method 1 (solve bracket first): $68 - (31) = 37$.

    Method 2 (remove brackets): $68 - 18 - 13 = 50 - 13 = 37$.

  • Algebraic expressions take number values when letter-numbers are replaced by numbers (e.g., $a + 3$ becomes $23 + 3 = 26$ when $a = 23$).
4.3 Omission of the Multiplication Symbol in Algebraic Expressions

This section explains the convention of omitting the multiplication symbol in algebraic expressions.

  • Number sequence: $4, 8, 12, 16, 20, 24, 28, ...$
  • This is the multiplication table of 4.
  • The $n^{th}$ term is $4 imes n$.
  • Standard practice: Shorten $4 imes n$ to $4n$ by skipping the multiplication sign. The number is written first, followed by the letter(s).
  • Examples:
    • Value of $7k$ when $k = 4$: $7 imes 4 = 28$.
    • Value of $5m + 3$ when $m = 2$: $5 imes 2 + 3 = 10 + 3 = 13$.

Mind the Mistake, Mend the Mistake

Identify and correct mistakes in simplifying expressions:

  • 1. If $a = -4$, then $10 - a = 6$.
    • Mistake: $10 - (-4) = 10 + 4 = 14$.
    • Correct value: $14$.
  • 2. If $d = 6$, then $3d = 36$.
    • Mistake: $3 imes 6 = 18$.
    • Correct value: $18$.
  • 3. If $s = 7$, then $3s - 2 = 15$.
    • Mistake: $3 imes 7 - 2 = 21 - 2 = 19$.
    • Correct value: $19$.
  • 4. If $r = 8$, then $2r + 1 = 29$.
    • Mistake: $2 imes 8 + 1 = 16 + 1 = 17$.
    • Correct value: $17$.
  • 5. If $j = 5$, then $2j = 10$.
    • Correct: $2 imes 5 = 10$.
  • 6. If $m = -6$, then $3(m + 1) = 19$.
    • Mistake: $3(-6 + 1) = 3(-5) = -15$.
    • Correct value: $-15$.
  • 7. If $f = 3, g = 1$, then $2f - 2g = 2$.
    • Mistake: $2 imes 3 - 2 imes 1 = 6 - 2 = 4$.
    • Correct value: $4$.
  • 8. If $t = 4, b = 3$, then $2t + b = 24$.
    • Mistake: $2 imes 4 + 3 = 8 + 3 = 11$.

4.4 Simplification of Algebraic Expressions

This section focuses on simplifying algebraic expressions by combining like terms.

Perimeter of a Rectangle

  • Perimeter = length + breadth + length + breadth.
  • Using letter-numbers $l$ for length and $b$ for breadth: $p = l + b + l + b$.
  • Rearranging terms: $p = l + l + b + b$.
  • Simplifying: $p = 2l + 2b$.
  • The expression $2l + 2b$ is the simplified form of $l + b + l + b$.
  • Example: If $l = 3, b = 4$, then $l + b + l + b = 3 + 4 + 3 + 4 = 14$, and $2l + 2b = 2 imes 3 + 2 imes 4 = 6 + 8 = 14$.

Example 5: Shop Selling Pencils and Erasers

Price per pencil is $c$, price per eraser is $d$.

  • Money earned from pencils:
    • Day 1: $5c$
    • Day 2: $3c$
    • Day 3: $10c$
  • Total money from pencils: $5c + 3c + 10c = (5 + 3 + 10)c = 18c$.
  • If $c = ‚āĻ50$, total earned from pencils: $18 imes 50 = ‚āĻ900$.
  • Money earned from erasers:
    • Day 1: $4d$
    • Day 2: $6d$
    • Day 3: $1d$
  • Total money from erasers: $4d + 6d + 1d = (4 + 6 + 1)d = 11d$.
  • Total money earned from pencils and erasers: $18c + 11d$.
  • This expression cannot be simplified further as it contains different letter-numbers.
  • Like terms: Terms involving the same letter-numbers (e.g., $5c, 3c, 10c$). They can be added and simplified.
  • Unlike terms: Terms with different letter-numbers (e.g., $18c, 11d$). They cannot be combined.

Example 6: Area of a Rectangle (Distributive Property)

A big rectangle is split into two smaller rectangles with side lengths $v$ and $4$, and $v$ and $3$.

  • Area of smaller rectangles: $4v$ and $3v$.
  • Area of bigger rectangle (sum of smaller areas): $4v + 3v$.
  • Area of bigger rectangle (direct calculation): $v imes (4 + 3) = 7v$.
  • Thus, $4v + 3v = 7v$, demonstrating the distributive property.
  • For a rectangle AEFD with sides $n$ and $(12 - 4)$: Area = $n imes (12 - 4) = 8n$.
  • Alternatively, Area = $12n - 4n = 8n$.

Example 7: Shop Renting Chairs and Tables

Rent: Chair ‚āĻ40, Table ‚āĻ75. Return: Chair ‚āĻ6, Table ‚āĻ10.

  • Total amount paid at beginning for $x$ chairs and $y$ tables: $40x + 75y$.
  • Total amount returned for $x$ chairs and $y$ tables: $6x + 10y$.
  • Net amount paid = $(40x + 75y) - (6x + 10y)$.
  • Simplifying the expression:
    • $(40x + 75y) - 6x - 10y$
    • Group like terms: $(40x - 6x) + (75y - 10y)$
    • Result: $(40 - 6)x + (75 - 10)y = 34x + 65y$.

Example 10: Comparing $5u$ and $5 + u$

  • $5u$ means 5 times $u$.
  • $5 + u$ means 5 more than $u$.
  • These expressions are generally not equal.
  • Example values:
    • If $u = 2$: $5u = 10$, $5 + u = 7$.
    • If $u = 5$: $5u = 25$, $5 + u = 10$.
    • If $u = 8$: $5u = 40$, $5 + u = 13$.
    • If $u = 11$: $5u = 55$, $5 + u = 16$.

Example 11: Comparing $10y - 3$ and $10(y - 3)$

  • $10y - 3$ means 3 less than 10 times $y$.
  • $10(y - 3)$ means 10 times (3 less than $y$).
  • These expressions are generally not equal.
  • Example values:
    • If $y = 0$: $10y - 3 = -3$, $10(y - 3) = -30$.
    • If $y = 2$: $10y - 3 = 17$, $10(y - 3) = -10$.
    • If $y = 7$: $10y - 3 = 67$, $10(y - 3) = 40$.
    • If $y = 10$: $10y - 3 = 97$, $10(y - 3) = 70$.
