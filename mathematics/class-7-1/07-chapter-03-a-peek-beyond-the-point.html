Class 7 (Part 1) — Chapter 3: A Peek Beyond the Point

3.1 The Need for Smaller Units

When measuring objects with high precision, standard units like centimeters may not be sufficient. We often encounter lengths that fall between two consecutive centimeter marks on a ruler, necessitating the use of smaller, subdivided units.

  • For example, a screw might have a length between $2\text{ cm}$ and $3\text{ cm}$.
  • By dividing the centimeter into $10$ equal parts, we can express this length more accurately as $2\text{ cm}$ and $7$ tenths of a centimeter, written as $2 \frac{7}{10}\text{ cm}$ (read as two and seven-tenth centimeters).
3.2 A Tenth Part

When a single unit is divided into $10$ equal parts, each part is called one-tenth of a unit, represented mathematically as $\frac{1}{10}$.

  • The length of an object, such as a pencil, can be written as $3 \frac{4}{10}$ units. This represents $3$ whole units and $4$ tenths of a unit: $$(3 \times 1) + \left(4 \times \frac{1}{10}\right)\text{ units}$$
  • Since $10$ tenths make $1$ whole unit ($10 \times \frac{1}{10} = 1$), the length $3 \frac{4}{10}$ is equivalent to $\frac{34}{10}$ units (or $34$ one-tenths).

Reading Fractional Units

  • $4 \frac{1}{10}$ is read as 'four and one-tenth'
  • $\frac{4}{10}$ is read as 'four one-tenths' or 'four-tenths'
  • $\frac{41}{10}$ is read as 'forty-one one-tenths' or 'forty-one tenths'
  • $41 \frac{1}{10}$ is read as 'forty-one and one-tenth'
3.3 A Hundredth Part

To measure even smaller lengths, we can divide each one-tenth part into $10$ equal subdivisions. Since there are $10$ tenths in a unit, dividing each tenth into $10$ parts yields $100$ equal parts in a single unit.

  • Each of these smaller parts represents one-hundredth of a unit, written as $\frac{1}{100}$.
  • A measurement ending at $4$ units, $4$ tenths, and $5$ hundredths is written as $4 \frac{4}{10} \frac{5}{100}$ (read as four units, four one-tenths, and five one-hundredths).
  • This can also be expressed as $4$ units and $45$ hundredths ($4 \frac{45}{100}$).
3.4 Decimal Place Value

The decimal system is a natural extension of our base-10 place value system to represent fractional parts. Each place value is $10$ times smaller than the one immediately to its left.

To separate the whole number (integer) part from the fractional part, we use a separator called the decimal point (represented by a period '.').

Place Value Table

The place values extend to the right of the ones place as follows:

  • Hundreds: $100$
  • Tens: $10$
  • Ones (Units): $1$
  • Tenths: $\frac{1}{10}$ or $0.1$
  • Hundredths: $\frac{1}{100}$ or $0.01$
  • Thousandths: $\frac{1}{1000}$ or $0.001$

Reading Decimal Numbers

  • $705$ is read as "seven hundred and five".
  • $70.5$ is read as "seventy point five" (short for seventy and five-tenths).
  • $7.05$ is read as "seven point zero five" (short for seven and five hundredths).
  • $0.274$ is read as "zero point two seven four" (representing $2$ tenths, $7$ hundredths, and $4$ thousandths). We do not read the fractional part as "two hundred and seventy-four".
3.5 Units of Measurement

Decimal notation simplifies conversions between different units of length, weight, and currency.

Length Conversion

  • Since $1\text{ cm} = 10\text{ mm}$, we have: $$1\text{ mm} = \frac{1}{10}\text{ cm} = 0.1\text{ cm}$$
  • Since $1\text{ m} = 100\text{ cm}$, we have: $$1\text{ cm} = \frac{1}{100}\text{ m} = 0.01\text{ m}$$
  • Since $1\text{ m} = 1000\text{ mm}$, we have: $$1\text{ mm} = \frac{1}{1000}\text{ m} = 0.001\text{ m}$$

Weight Conversion

  • Since $1\text{ kg} = 1000\text{ g}$, we have: $$1\text{ g} = \frac{1}{1000}\text{ kg} = 0.001\text{ kg}$$
  • Since $1\text{ g} = 1000\text{ mg}$, we have: $$1\text{ mg} = \frac{1}{1000}\text{ g} = 0.001\text{ g}$$

Rupee-Paise Conversion

  • Since $1\text{ Rupee} = 100\text{ paise}$, we have: $$1\text{ paisa} = \frac{1}{100}\text{ Rupee} = \text{₹ } 0.01$$
  • $75\text{ paise} = \frac{75}{100}\text{ Rupee} = \text{₹ } 0.75$
3.6 Locating and Comparing Decimals

Locating on a Number Line

To locate a decimal number like $1.4$ on a number line, we identify that it lies between $1$ and $2$. We divide the unit interval between $1$ and $2$ into $10$ equal parts and count $4$ parts from $1$ to find $1.4$.

The Zero Dilemma

Appending zeros to the rightmost end of a decimal number does not change its value. For example: $$0.2 = 0.20 = 0.200$$ This is because they all represent the same quantity ($2$ tenths). However, $0.2$, $0.02$, and $0.002$ are entirely different values because the digit $2$ occupies different place values.

Comparing Decimals

To compare two decimal numbers:

  1. Compare the digits starting from the highest place value (left to right).
  2. If the digits are the same, move to the next place value to the right.
  3. The number with the larger digit at the first point of difference is the greater number.
  4. For example, comparing $6.456$ and $6.465$: both have $6$ units and $4$ tenths, but the second number has $6$ hundredths while the first has only $5$ hundredths. Thus, $6.465 > 6.456$.
3.7 Addition and Subtraction of Decimals

Decimals are added and subtracted using the same standard vertical algorithms as whole numbers, with one critical rule: align the decimal points vertically.

  • This alignment ensures that digits of the same place value (tens with tens, ones with ones, tenths with tenths, etc.) are added or subtracted together.
  • If necessary, write trailing zeros to make the numbers have the same number of decimal places.
  • Example Addition: $$75.345 + 86.691 = 162.036$$
  • Example Subtraction: $$3.5 - 2.7 = 0.8$$
3.8 More on the Decimal System

Deceptive Decimal Notation

Sometimes, decimal-like notations are used in non-decimal contexts, which can lead to confusion:

  • Time: $4.5\text{ hours}$ does not mean $4\text{ hours and } 5\text{ minutes}$. Since $0.5$ represents half of an hour, it equals $30\text{ minutes}$. Thus, $4.5\text{ hours} = 4\text{ hours and } 30\text{ minutes}$.
  • Cricket: An over count of $5.5$ overs means $5$ overs and $5$ balls (where $1\text{ over} = 6\text{ balls}$), which is mathematically $5 \frac{5}{6}$ overs, not a true decimal.

History of Decimal Notation

Decimal fractions were used by ancient Indian astronomers and mathematicians, including the 8th-century scholar Śhrīdharāchārya. The system was detailed in the 10th century by Arab mathematician Abūl Ḥassan al-Uqlīdisī. In the 16th century, mathematicians like John Napier popularized the use of the point/period ('.') as a decimal separator.
