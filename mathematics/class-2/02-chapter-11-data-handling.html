Class 2 — Chapter 11: Data Handling

11.1 Favourite Colours

In this section, we learn how to collect, organize, and represent data about children's favorite colours using a simple table.

Activity: Collecting Colour Data

Look at the picture of children in the park and count how many children prefer each colour to complete the table:

  • Red: Count the children wearing red.
  • Green: Count the children wearing green.
  • Blue: Count the children wearing blue.
  • Yellow: Count the children wearing yellow.

Data Analysis Questions

Answer the following questions based on the collected data:

  • The most liked colour is the one with the highest count.
  • The least liked colour is the one with the lowest count.
  • Comparing preferences: "Yellow colour is liked more than [Colour]" and "[Colour] is liked less than [Colour]".
11.2 Picnic Day (Fruit Party)

Children are having a fruit party and eating their favourite fruits. We represent this information in a table to analyze preferences.

Activity: Fruit Count Table

Count the number of children eating each of the following fruits:

  • Banana
  • Orange
  • Apple
  • Kiwi
  • Guava

Analysis and Comparison

Use the table to answer questions using comparison terms like more than, less than, or equal to:

  • Total number of children in the picture.
  • Most liked and least liked fruits.
  • Comparing the number of children who like guava vs. apple, apple vs. banana, and orange vs. kiwi.
11.3 Mode of Coming to School

This section introduces representing data using simple pictographs (drawing faces to represent students).

Activity: Mode of Transport Table

Count the students using different modes of transport to reach school from the picture:

  • On-foot: $8$ students
  • Autorickshaw: Count from picture
  • Bicycle: Count from picture
  • Bike: Count from picture
  • School Bus: Count from picture

Drawing a Pictograph

Draw faces ($\odot$) in the chart where $\odot = 1$ student to represent the data visually.

  • Analyze which mode is used by the most and least number of students.
  • Compare transport modes using "more than" or "less than".
11.4 Games We Play

Representing and analyzing data about outdoor games played by children.

Activity: Games Count

Count the number of students playing each game in the picture:

  • Cricket
  • Football
  • Pithu

11.5 Survey and Project Work

Practical application of data handling through surveys and real-world data collection.

Class Survey: Favourite Vegetables

Ask classmates about their favourite vegetables, record the counts in a table, and identify the most and least liked vegetables.

Family Size Survey

Collect data on the number of family members living in your friends' homes:

  • Count families with $1$, $2$, $3$, $4$, $5$, or more people.
  • Identify the most common family size.

Project Work: Grandparents

Visit nearby families to find out how many live with:

  • Both grandparents
  • Only female grandparent
  • Only male grandparent
  • No grandparents
11.6 Puzzles and Brain Teasers

A collection of logical, mathematical, and visual puzzles to develop critical thinking.

Visual and Path Puzzles

  • Shape Connection: Connect pairs of similar shapes without crossing lines or using diagonals.
  • Ant's Path: Find different ways for an ant to return to its anthill.
  • Cat's Path: Determine the number of different paths a cat can take to reach its house.
  • Giraffe's Maze: Draw a path for a hungry giraffe to reach the tree, noting which animals it meets first and last.

Mathematical and Number Puzzles

  • Pattern Completion: Complete the intersecting addition equations:
    • $2 + \text{blank} = 4$
    • $\text{blank} + 1 = \text{blank}$
    • $1 + 3 = \text{blank}$
  • Math Magic Trick:
    1. Think of a number ($x$).
    2. Double it ($2x$).
    3. Add eight ($2x + 8$).
    4. Minus half of that number ($(2x + 8) - (x + 4) = x + 4$).
    5. Minus the number you started with ($(x + 4) - x = 4$).
    6. The result is always $4$!
  • Number Search: Find the number names from $11$ to $20$ in the word grid.
  • Ball Selection: Select three balls from $\{1, 3, 5, 7, 9, 11, 13, 15\}$ to make a total of $30$.
  • Two-Digit Number: Make two different two-digit numbers using only one digit (e.g., $11$, $22$, etc.).
  • Double Number Puzzle: "I think of a number and double it. If my answer is $18$, then what was the number?" (Solve: $2x = 18 \implies x = 9$).
  • Calculator Challenge: Press exactly $10$ keys on a calculator to make the display show $28$.

Logical and Spatial Puzzles

  • Shadow Matching: Identify the correct shadow image for the multi-armed bug.
  • Matchstick Puzzle: Remove exactly two matchsticks from a grid of four squares to leave exactly two squares.
  • Riddle: "I am in a bicycle, and in a car and a bus, I make all vehicles run, they can't move if I am flat. What is my shape?" (Answer: Circle / Wheel).
  • Rectangle Counting: Find how many rectangles (including squares) can be made by joining a $3 \times 3$ grid of dots.
  • Hexagon Math: Complete the hexagon patterns using addition or subtraction to reach the center numbers ($9$, $13$, $19$).
  • Shepherd's Garden: Divide a circular garden containing $4$ sheep, $4$ dogs, and $4$ trees into $4$ equal areas such that each area contains exactly one tree, one sheep, and one dog.
  • Mirror Images: Use a mirror to read the reversed numbers (e.g., $42$, $38$, $e9$, etc.).
  • Arm Sum Puzzle: Arrange numbers $1$ to $7$ in a three-armed diagram so that all three arms have the same sum.
  • Cloth Cutting: "Let us divide a $6$ meter long cloth into pieces. If one meter is cut every time, how many times should we cut it?" (Answer: $5$ cuts).
