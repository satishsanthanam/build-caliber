Chapter 8: Predicting What Comes Next: Exploring Sequences and Progressions

8.1 Introduction to Sequences
  • A sequence is an ordered list of numbers where each number is called a term.
  • Sequences can be:
    • Finite sequence: Has a limited number of terms (e.g., 6, 12, 24, 48, 96).
    • Infinite sequence: Continues indefinitely, indicated by three dots (...) (e.g., 1, 2, 3, 4, ...).
  • Notation: Terms are represented as t1, t2, t3, ..., tn, where the subscript denotes the position of the term.
8.2 Explicit Rule for a Sequence
  • An explicit formula uses the term's position number, n, to calculate its value directly without needing previous terms.
  • Example: For the sequence of odd numbers (1, 3, 5, ...), the explicit rule is un = 2n - 1.
8.3 Recursive Rule for a Sequence
  • A recursive rule (or formula) defines a term by relating it to one or more previous terms. It requires specifying the initial term(s).
  • Example: t1 = 1, tn = tn-1 + 3 for n ≥ 2 generates the sequence 1, 4, 7, 10, ...
  • The Virahānka-Fibonacci sequence is a famous recursive sequence:
    • Rule: V1 = 1, V2 = 2, Vn = Vn-1 + Vn-2 for n ≥ 3.
    • Sequence: 1, 2, 3, 5, 8, 13, 21, 34, ...
    • Historical context: First documented by Virahānka (7th century CE) in the context of poetic meters, and later studied by Gopāla (1135 CE), Hemachandra (1150 CE), and Fibonacci (1200 CE).
8.4 Arithmetic Progressions (APs)
  • An arithmetic progression is a sequence in which the difference between any two consecutive terms is constant. This constant is called the common difference (d).
  • The first term is denoted by a.
  • General Form of an AP: a, a + d, a + 2d, a + 3d, ..., a + (n - 1)d
  • Explicit Formula (nth term): tn = a + (n - 1)d
  • Recursive Formula: t1 = a, tn = tn-1 + d for n ≥ 2
8.5 Sum of the First n Natural Numbers
  • The sum of the first n natural numbers (Sn = 1 + 2 + 3 + ... + n) is given by:
    Sn = n(n + 1) / 2
  • Historical Note: This result was first written in India by Āryabhaṭa in his work Āryabhaṭīya (Chapter 2, Verse 19).
8.6 Geometric Progressions (GPs)
  • A geometric progression is a sequence where each term after the first is obtained by multiplying the preceding term by a non-zero constant called the common ratio (r).
  • General Form of a GP: a, ar, ar2, ar3, ..., arn-1
  • Explicit Formula (nth term): tn = a · rn-1
  • Recursive Formula: t1 = a, tn = r · tn-1 for n ≥ 2
  • Fractals: Shapes or patterns that repeat themselves at different scales (e.g., the Sierpiński triangle).
    • For a Sierpiński triangle starting with area 1:
      • Number of black triangles at stage n: tn = 3n
      • Area of black region at stage n: sn = (3/4)n
