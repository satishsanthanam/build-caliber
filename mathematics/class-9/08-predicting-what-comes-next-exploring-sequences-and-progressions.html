Chapter 8: Predicting What Comes Next: Exploring Sequences and Progressions

8.1 Introduction to Sequences
  • A sequence is an ordered list of numbers where each number is a term.
  • Sequences can be finite (e.g., 6, 12, 24, 48, 96) or infinite (e.g., natural numbers, odd numbers).
  • Common number patterns include Natural, Odd, Triangular, and Square numbers.
8.2 Explicit Rule for a Sequence
  • An explicit formula uses the term's position number, n, to calculate its value directly.
  • Notation: tn (or sn, un) represents the term in the nth position.
  • Allows finding the value of any term without knowing previous terms.
8.3 Recursive Rule for a Sequence
  • A recursive rule (or recursive formula) describes a sequence by relating terms to previous terms.
  • Requires knowing earlier terms to find the next terms.
  • The Virahānka–Fibonacci sequence is formed by adding the previous two terms: V1 = 1, V2 = 2, Vn = Vn-1 + Vn-2 for n ≥ 3.
8.4 Arithmetic Progressions
  • An arithmetic progression (AP) is a sequence where the difference between consecutive terms is constant.
  • The first term is denoted by a.
  • The common difference (d) is the fixed number added to the previous term.
  • General form: a, a + d, a + 2d, a + 3d, ..., a + (n - 1)d.
  • Explicit rule (nth term): tn = a + (n - 1)d.
  • Recursive rule: t1 = a, tn = tn-1 + d for n ≥ 2.
  • Visualizing an AP: Plotting term position (x) vs. term value (y) forms a straight line.
8.5 Sum of the First n Natural Numbers
  • Formula for the sum of the first n natural numbers: Sn = n(n + 1) / 2.
  • The nth term of the triangular number sequence equals the sum of the first n natural numbers.
8.6 Geometric Progressions
  • A geometric progression (GP) is a sequence where each term after the first is obtained by multiplying the previous term by a constant multiplier.
  • The common ratio (r) is the constant multiplier between consecutive terms.
  • General form: a, ar, ar², ar³, ..., ar^(n-1).
  • Explicit rule (nth term): tn = a * r^(n-1).
  • Recursive rule: t1 = a, tn = r * tn-1 for n ≥ 2.
  • Visualizing a GP: Plotting term position vs. term value does not form a straight line (forms an exponential curve).
  • Fractals are shapes or patterns that repeat themselves at different scales (e.g., Sierpiński triangle). Their attributes often lead to geometric progressions.
Chapter Summary
  • Sequence: an ordered list of numbers.
  • Explicit formula: generates a term using its position number, n.
  • Recursive formula: generates a term using the values of previous terms.
  • Triangular number sequence: nth term is tn = n(n + 1) / 2.
  • Arithmetic progression (AP): tn = a + (n - 1)d.
  • Geometric progression (GP): tn = a * r^(n-1).
