8.1 Introduction to Sequences

We see patterns around us everywhere, in nature, art, music, finance, and everyday life. In mathematics, a sequence is an ordered list of numbers where each number is a term of the sequence.

Sequences can be finite or infinite. Examples of infinite sequences include:

  • Natural Numbers: $1, 2, 3, 4, 5, 6, \dots$
  • Odd Numbers: $1, 3, 5, 7, 9, 11, \dots$
  • Triangular Numbers: $1, 3, 6, 10, 15, 21, \dots$ (where each term is the sum of natural numbers up to that term, e.g., $15 = 1 + 2 + 3 + 4 + 5$)
  • Square Numbers: $1, 4, 9, 16, 25, 36, \dots$ (where each term is the sum of consecutive odd numbers, e.g., $16 = 1 + 3 + 5 + 7$)

Notation: We use $t_1$ to represent the first term, $t_2$ for the second term, and $t_n$ for the $n^{\text{th}}$ term. Other letters like $s_n$ or $u_n$ can also be used to represent terms of different sequences. The subscript $n$ is always a positive integer, while the term itself can be any real number.

8.2 Explicit Rule for a Sequence

An explicit formula uses the term's position number, $n$, to calculate its value directly without needing to know the previous terms.

  • Example 1: The explicit rule for the sequence of odd numbers is $u_n = 2n - 1$. For $n = 3$, $u_3 = 2(3) - 1 = 5$.
  • Example 2: For the sequence generated by $s_n = 5n - 2$, the $100^{\text{th}}$ term is $s_{100} = 5(100) - 2 = 498$. We can check if a number like $308$ is a term by solving $5n - 2 = 308 \implies 5n = 310 \implies n = 62$ (since $62$ is a natural number, $308$ is the $62^{\text{nd}}$ term).
8.3 Recursive Rule for a Sequence

A recursive rule or recursive formula defines a term of a sequence by relating it to one or more of the previous terms. To use a recursive rule, the initial term(s) must be specified.

  • Example: The sequence $1, 4, 7, 10, 13, \dots$ can be written recursively as $t_1 = 1$ and $t_n = t_{n-1} + 3$ for $n \ge 2$.
  • Virahānka–Fibonacci Sequence: A famous recursive sequence where each term is the sum of the preceding two terms: $$V_1 = 1, V_2 = 2, V_n = V_{n-1} + V_{n-2} \text{ for } n \ge 3$$ This generates the sequence: $1, 2, 3, 5, 8, 13, 21, 34, \dots$. It was first written down by the Indian scholar Virahānka in the 7th century CE in the context of Sanskrit/Prakrit poetic meters, and later studied by Gopāla, Hemachandra, and Fibonacci.
8.4 Arithmetic Progressions

An arithmetic progression (AP) is a sequence in which the difference between any two consecutive terms is a constant. This constant difference is called the common difference ($d$).

In general, an AP can be written as: $$a, a + d, a + 2d, a + 3d, \dots, a + (n - 1)d$$ where $a$ is the first term and $d$ is the common difference.

  • Explicit Formula for the $n^{\text{th}}$ Term: $$t_n = a + (n - 1)d$$
  • Recursive Rule: $$t_1 = a, t_n = t_{n-1} + d \text{ for } n \ge 2$$

8.4.1 Visualising an AP

When we plot the ordered pairs $(n, t_n)$ representing the term number and its value for an arithmetic progression, the points lie on a straight line. This indicates that an AP represents a linear pattern.

8.5 Sum of the First n Natural Numbers

The sum of the first $n$ natural numbers, denoted by $S_n = 1 + 2 + 3 + \dots + n$, is given by the formula: $$S_n = \frac{n(n + 1)}{2}$$

Derivation: By writing the sum in forward and reverse order and adding them term-by-term: $$\begin{aligned} S_n &= 1 + 2 + \dots + n \\ S_n &= n + (n-1) + \dots + 1 \\ \hline 2S_n &= (n+1) + (n+1) + \dots + (n+1) \quad [n \text{ times}] \\ 2S_n &= n(n+1) \\ S_n &= \frac{n(n+1)}{2} \end{aligned}$$

Historical Note: This result was first recorded in India by Āryabhaၣa in his work *Āryabhaၣĩya* (c. 499 CE), Chapter 2, Verse 19.

8.6 Geometric Progressions

A geometric progression (GP) is a sequence in which each term after the first is obtained by multiplying the previous term by a non-zero constant called the common ratio ($r$).

In general, a GP can be written as: $$a, ar, ar^2, ar^3, \dots, ar^{n-1}$$ where $a$ is the first term and $r$ is the common ratio.

  • Explicit Formula for the $n^{\text{th}}$ Term: $$t_n = a r^{n-1}$$
  • Recursive Rule: $$t_1 = a, t_n = r t_{n-1} \text{ for } n \ge 2$$

8.6.1 Fun with Fractals

Fractals are self-similar geometric patterns that repeat at different scales. A classic example is the Sierpiński triangle:

  • Number of Black Triangles: At stage $n$, the number of black triangles is given by $t_n = 3^n$, which forms a GP: $1, 3, 9, 27, \dots$
  • Area of Black Region: If the initial area is $1$ square unit, the area at stage $n$ is $s_n = \left(\frac{3}{4}\right)^n$, which forms a GP: $1, \frac{3}{4}, \frac{9}{16}, \dots$

8.6.2 Visualising a GP

When we plot the ordered pairs $(n, t_n)$ for a geometric progression, the points do not lie on a straight line. Instead, they form a curve representing exponential growth (if $r > 1$) or exponential decay (if $0 < r < 1$).

Chapter Summary
  • A sequence is an ordered list of numbers where each number is a term.
  • An explicit formula calculates a term's value directly from its position $n$.
  • A recursive formula defines a term using the values of previous terms.
  • The triangular number sequence is given by $t_n = \frac{n(n+1)}{2}$, which is also the sum of the first $n$ natural numbers.
  • An arithmetic progression (AP) has a constant difference between consecutive terms: $t_n = a + (n-1)d$.
  • A geometric progression (GP) has a constant ratio between consecutive terms: $t_n = a r^{n-1}$.
  • Fractals like the Sierpiński triangle and Sierpiński square carpet demonstrate geometric progressions in their structural properties.
