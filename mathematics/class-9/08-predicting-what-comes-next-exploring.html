Chapter 8: Predicting What Comes Next: Exploring Sequences and Progressions

8.1 Introduction to Sequences
  • A sequence is an ordered list of numbers where each number is a term .
  • Sequences can be finite (having a specific number of terms) or infinite (continuing indefinitely).
  • Examples include natural numbers, odd numbers, triangular numbers, and square numbers.
8.2 Explicit Rule for a Sequence
  • An explicit formula uses a term's position number, n, to calculate its value directly.
  • Notation typically uses subscripts matching the term numbers, such as tn, sn, or un for the nth term.
  • Allows finding the value of any term without knowing previous terms.
8.3 Recursive Rule for a Sequence
  • A recursive rule (or recursive formula) describes a sequence by relating terms to one or more previous terms.
  • Requires knowing earlier terms to find subsequent terms.
  • The Virahanka-Fibonacci sequence is a famous recursive sequence where each term is the sum of the previous two: V1 = 1, V2 = 2, and Vn = Vn-1 + Vn-2 for n >= 3.
8.4 Arithmetic Progressions
  • An arithmetic progression (AP) is a sequence in which the difference between consecutive terms is a constant value.
  • This constant value is called the common difference (d).
  • The first term is denoted as a.
  • General form of an AP: a, a + d, a + 2d, a + 3d, ..., a + (n - 1)d.
  • Explicit formula for the nth term: tn = a + (n - 1)d.
  • Recursive rule for an AP: t1 = a, tn = tn-1 + d for n >= 2.
  • Plotting the terms of an AP against their position numbers results in points that lie on a straight line.
8.5 Sum of the First n Natural Numbers
  • The sum of the first n natural numbers is given by the formula: Sn = n(n + 1) / 2.
  • This formula can also be used to find the nth term of the triangular number sequence.
  • The first known written mention of this result is found in Aryabhata's Aryabhatiya.
8.6 Geometric Progressions
  • A geometric progression (GP) is a sequence where each term after the first is obtained by multiplying the previous term by a constant multiplier.
  • This constant multiplier is known as the common ratio (r).
  • General form of a GP: a, ar, ar^2, ar^3, ..., ar^(n-1).
  • Explicit formula for the nth term: tn = a * r^(n-1).
  • Recursive rule for a GP: t1 = a, tn = r * tn-1 for n >= 2.
  • Fractals , such as the Sierpinski triangle and Sierpinski square carpet, are shapes that repeat themselves at different scales and often generate geometric progressions in their properties.
  • Points emerging from a GP do not lie on a straight line when plotted against their stage numbers.
Chapter Summary
  • A sequence is an ordered list of numbers called terms.
  • An explicit formula generates each term using its position number, n.
  • A recursive formula gives the value of a term using previous terms.
  • The triangular number sequence nth term is tn = n(n + 1) / 2.
  • An AP has a common difference d, with nth term tn = a + (n - 1)d.
  • A GP has a common ratio r, with nth term tn = a * r^(n-1).
