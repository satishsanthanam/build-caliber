Chapter 8: Predicting What Comes Next: Exploring Sequences and Progressions

8.1 Introduction to Sequences
  • A sequence is an ordered list of numbers where each number is a term .
  • Sequences can be finite or infinite.
  • Examples include Natural Numbers, Odd Numbers, Triangular Numbers, and Square Numbers.
8.2 Explicit Rule for a Sequence
  • An explicit formula uses the term's position number, n, to calculate its value.
  • Notation: tn (or sn, un) represents the term in the nth position.
  • Allows finding the value of any term directly without knowing previous terms.
8.3 Recursive Rule for a Sequence
  • A recursive rule (or recursive formula) describes a sequence by relating terms to previous terms.
  • Requires knowing earlier terms to find the next terms.
  • Virahānka-Fibonacci sequence : A sequence where each term is obtained by adding the previous two terms (V1 = 1, V2 = 2, Vn = Vn-1 + Vn-2 for n ≥ 3).
8.4 Arithmetic Progressions
  • An arithmetic progression (AP) is a sequence in which the difference between consecutive terms is constant.
  • The fixed difference is called the common difference (d).
  • The first term is denoted by a.
  • General form of an AP: a, a + d, a + 2d, a + 3d, ..., a + (n - 1)d.
  • Explicit rule for the nth term: tn = a + (n - 1)d.
  • Recursive rule: t1 = a, tn = tn-1 + d for n ≥ 2.
  • Visualising an AP: Plotting the stage number against the term value results in points that lie on a straight line.
8.5 Sum of the First n Natural Numbers
  • The sum of the first n natural numbers is given by the formula: Sn = n(n + 1) / 2.
  • This formula also represents the nth term of the triangular number sequence.
8.6 Geometric Progressions
  • A geometric progression (GP) is a sequence in which each term after the first is obtained by multiplying the previous term by a constant multiplier.
  • The constant multiplier is called the common ratio (r).
  • General form of a GP: a, ar, ar², ar³, ..., ar^(n-1).
  • Explicit rule for the nth term: tn = a * r^(n-1).
  • Recursive rule: t1 = a, tn = r * tn-1 for n ≥ 2.
  • Fractals (like the Sierpiński triangle) are shapes or patterns that repeat themselves at different scales, often generating geometric progressions in their properties.
  • Visualising a GP: Plotting the stage number against the term value results in points that form a curve, not a straight line.
Chapter Summary
  • A sequence is an ordered list of numbers; each is a term.
  • Explicit formula generates each term using its position number n.
  • Recursive formula gives the value of a term using previous terms.
  • Triangular number sequence nth term: tn = n(n + 1) / 2 (also the sum of first n natural numbers).
  • AP: Each term is obtained by adding a fixed common difference d to the previous term. nth term: tn = a + (n - 1)d.
  • GP: Each term is obtained by multiplying the previous term by a fixed common ratio r. nth term: tn = a * r^(n-1).
