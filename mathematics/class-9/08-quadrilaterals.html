8.1 Introduction to Sequences
  • A sequence is an ordered list of numbers.
  • Each number in the sequence is called a term.
  • Sequences can be finite (e.g., 6, 12, 24, 48, 96) or infinite (e.g., odd numbers).
8.2 Explicit Rule for a Sequence
  • An explicit formula uses the term's position number, n, to calculate its value directly.
  • Example: t_n = 2n - 1 for the sequence of odd numbers.
8.3 Recursive Rule for a Sequence
  • A recursive rule (or formula) describes a sequence by relating terms to previous terms.
  • Requires knowing earlier terms to find the next terms.
  • Example: t_1 = 1, t_n = t_{n-1} + 3.
  • Virahānka-Fibonacci sequence: A sequence where each term is obtained by adding the previous two terms. V_1 = 1, V_2 = 2, and V_n = V_{n-1} + V_{n-2} for n ≥ 3.
8.4 Arithmetic Progressions (AP)
  • An arithmetic progression is a sequence in which the difference between consecutive terms is constant.
  • The constant difference is called the common difference (d).
  • The first term is denoted by a.
  • General form of an AP: a, a + d, a + 2d, a + 3d, ..., a + (n - 1)d.
  • The n-th term is given by the explicit formula: t_n = a + (n - 1)d.
  • Recursive rule for an AP: t_1 = a, t_n = t_{n-1} + d for n ≥ 2.
8.5 Sum of the First n Natural Numbers
  • The sum of the first n natural numbers is given by: S_n = n(n + 1) / 2.
  • The n-th term of the triangular number sequence (1, 3, 6, 10, 15...) is the sum of the first n natural numbers.
8.6 Geometric Progressions (GP)
  • A geometric progression is a sequence in which each term after the first is obtained by multiplying the previous term by a constant multiplier.
  • This constant multiplier is called the common ratio (r).
  • General form of a GP: a, ar, ar^2, ar^3, ..., ar^{n-1}.
  • The n-th term is given by: t_n = ar^{n-1}.
  • Fractals: Shapes or patterns that repeat themselves at different scales (e.g., Sierpiński triangle). Many attributes of fractals lead to geometric progressions.
Chapter Summary
  • A sequence is an ordered list of numbers; each number is a term.
  • A general formula generates each term. An explicit formula uses position n.
  • A recursive formula uses values of previous terms.
  • Triangular number sequence: t_n = n(n + 1) / 2.
  • Arithmetic progression (AP): t_n = a + (n - 1)d.
  • Geometric progression (GP): t_n = ar^{n-1}.
