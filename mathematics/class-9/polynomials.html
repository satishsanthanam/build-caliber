Chapter 2: Introduction to Linear Polynomials

2.1 Introduction to Polynomials

Anatomy of an Algebraic Expression
  • Variables: Letter-numbers (e.g., x, y, z) representing unknown values.
  • Terms: Parts of an expression separated by addition or subtraction (e.g., in 4x + 5y + 3, the terms are 4x, 5y, and 3).
  • Coefficients: The numerical factors of the variable terms (e.g., 4 and 5 are coefficients of x and y).
  • Constant: A term with a fixed value containing no variables (e.g., 3).

Univariate Polynomials & Degree
  • Univariate Polynomials: Algebraic expressions involving only one variable (e.g., x, y, or z) with non-negative integer exponents.
  • Degree: The highest power of the variable in a polynomial.
  • Classification by Degree:
    • Constant Polynomials: Degree 0 (e.g., 8, which can be written as 8x0).
    • Linear Polynomials: Degree 1 (e.g., 3z + 7).
    • Quadratic Polynomials: Degree 2 (e.g., x2 + 5x + 1).
    • Cubic Polynomials: Degree 3 (e.g., 5y3 + y2 + 2y - 1).
2.2 Linear Polynomials & Equations

Linear Patterns & Equations
  • Linear Patterns: Sequences where the difference between successive values at integer intervals is constant.
  • Linear Equation: Formed when a linear polynomial in one variable is equated to a constant (e.g., 2x + 10 = 64).
  • Function: An input-output process where each input value of x yields a unique output value (e.g., f(x) = 2x + 3).
2.3 Exploring Linear Patterns

Generalising Sequences
  • A growing pattern of square tiles (1, 3, 5, 7...) can be generalised using the linear expression 2n - 1, where n is the stage number.
  • The constant difference of 2 between consecutive terms represents the rate of change of the pattern.
2.4 Linear Growth and Linear Decay

Growth vs. Decay
  • Linear Growth: A pattern where a quantity increases by a constant amount over equal intervals (e.g., C(d) = 100 + 60d).
  • Linear Decay: A pattern where a quantity decreases by a constant amount over equal intervals (e.g., h(t) = 3 - 0.5t).
2.5 Linear Relationships

The General Form
  • A linear relationship between two variables x and y is expressed as:
    y = ax + b
  • To find the constants a and b, substitute two known coordinate pairs (x1, y1) and (x2, y2) into the equation and solve the resulting system of simultaneous equations.
2.6 Visualising Linear Relationships

Slope and Intercept
  • Slope (a): Represents the steepness and direction of the line.
    • If a > 0, the line rises from left to right (positive slope / linear growth).
    • If a < 0, the line falls from left to right (negative slope / linear decay).
  • y-intercept (b): The point (0, b) where the line cuts the y-axis.
  • Graphical Behavior:
    • Changing a while keeping b fixed rotates the line around the y-intercept.
    • Changing b while keeping a fixed shifts the line vertically, creating parallel lines.
Chapter Summary
  • An algebraic expression combines numbers, variables, and operations.
  • Univariate polynomials contain only one variable. The highest power is the degree.
  • A polynomial of degree 1 is a linear polynomial.
  • Linear relationships are represented graphically by a straight line of the form y = ax + b, where a is the slope and b is the y-intercept.
